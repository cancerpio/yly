# Marumaru 唱歌學日語 - 學習機制分析與 Yomi Cards 功能改善建議

此文件記錄了對著名日語學習網站 [Marumaru 唱歌學日語](https://www.marumaru-x.com/) 的功能分析，並提出能為我們目前的 `Yomi Cards` 專案借鏡與強化的功能點。

## 🎯 Marumaru 核心優點分析

Marumaru 成功讓使用者黏著的原因在於**極致的「沉浸式音樂學習」體驗**，其核心優勢可以歸納為以下幾點：

1. **視覺層次分明的歌詞顯示 (Furigana / Ruby)**
   - 漢字上方會精準標示假名拼音（振假名 Furigana）。
   - 下方會提供羅馬拼音（Romaji）與中文翻譯。
   - 相比於我們目前單純將 `原文`、`羅馬音`、`中文` 分成三行，Marumaru 的立體排版（漢字與拼音結合）更能幫助大腦將漢字字碼與發音連結。

2. **時間軸同步顯示 (KTV 模式 / 同步滾動)**
   - 歌詞會隨著 YouTube 原曲的播放進度自動滾動、變色標示當前演唱的句子。
   - 解決了使用者不熟悉旋律時「不知道現在唱到哪裡」的痛點。

3. **高度客製化的學習輔助開關 (Toggles)**
   - 使用者可以自由開關：漢字、假名、羅馬拼音、中文翻譯。
   - 這是**非常重要的語言學習機制**（例如：想要挑戰聽力時，就把中文和假名關掉；想要背單字時，只看原文）。

4. **單句精細操作 (A-B Repeat & 回播)**
   - 每句歌詞旁邊都有一個「單句播放」的按鈕，點擊後影片會跳轉到該句並播放。
   - 有重複播放某一句的學習模式，針對發音難點進行突破。

5. **單字與文法解析 (Dictionary Integration)**
   - 滑鼠移到或點擊特定的單字，會彈出該字的字典解釋、詞性變化或語法說明。

## 🛠 Yomi Cards 專案改善與升級建議

考量到我們目前的 `Yomi Cards` 已經具備了強大的「即時分析 (MyMemory 翻譯 + Kuroshiro 羅馬音)」能力與「卡片清單」系統，我們可以借鏡 Marumaru，將專案從「單純的翻譯工具」升級為「強大的學習輔助器」。

### 第一階段：視覺與體驗優化 (Quick Wins)

1. **實作 Furigana (振假名) 渲染**
   - **現狀**：Kuroshiro 目前只輸出純羅馬拼音（`mode: "spaced"`）。
   - **改善**：Kuroshiro 其實支援輸出 `<ruby>` HTML 標籤（`mode: "furigana"`）。我們可以讓分析結果直接帶有振假名的 HTML，並在卡片列表 (PlaylistDetail) 中以更美觀的立體方式呈現，而不只是平行的三行文字。

2. **實作「學習模式」的遮罩開關**
   - **改善**：在 PlaylistDetail 的標題列新增「羅馬音」、「中文翻譯」的 Toggle 按鈕（眼睛圖示）。
   - **用途**：讓使用者可以一鍵隱藏所有的中文翻譯，強迫自己純看日文回想意思，達到真正的「閃卡（Flashcard）」學習效果。這是最低成本卻極大提升學習價值的改動。

### 第二階段：媒體與音樂整合 (Feature Expansion)

1. **整合 YouTube 播放器**
   - **現狀**：我們有歌名和歌手（透過 iTunes API 抓取）。
   - **改善**：在分析完成或進入 Playlist 頁面時，利用 YouTube Data API 自動嵌入該首歌的 MV。讓使用者能邊聽邊看卡片。

2. **單句語音發音 (TTS)**
   - **現狀**：卡片只有文字，沒有聲音。
   - **改善**：為每張卡片加上一個「喇叭」Icon，點擊後透過瀏覽器內建的 `Web Speech API` (SpeechSynthesis) 念出該句日文發音，幫助使用者校正語音。

### 第三階段：深度學習輔助 (Advanced)

1. **智慧斷詞與單字卡化**
   - **現狀**：我們的最小單位是「一句歌詞」。
   - **改善**：利用 Kuroshiro 底層的 Kuromoji 斷詞引擎，把一句歌詞拆成一個個獨立單字。點擊單字就可以獨立把這個字另外存成「單字清單」去背誦。

2. **時間軸同步 (Lyrics Syncing)**
   - **現狀**：靜態文字。
   - **改善**：提供一個介面，讓使用者可以邊播 YouTube 邊按空白鍵，為每一段歌詞打上時間標籤 (LRC 格式)。之後播放時就能像 Marumaru 一樣動態高亮當下正在唱的句子。

## 總結
Yomi Cards 的核心價值在於**「使用者能自己創造學習內容」**，這是 Marumaru 這種純提供固定曲庫的網站無法做到的（Marumaru 只能找他們建好的歌）。

如果我們能結合 **Marumaru 的優良展示介面（Furigana、隱藏遮罩、發音）** + **Yomi Cards 自動化且自由的批次建立能力**，這將會成為一個殺手級的客製化語言學習工具！
